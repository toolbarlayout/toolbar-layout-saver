Installation

Up and running
in 60 seconds.

1

Download the .rbz

From GitHub or Extension Warehouse

2

Install Extension

Extensions → Extension Manager → Install Extension

3

Save your layout

View → Toolbar Layout → Save Layout

4

Exit via File menu

File → Exit SketchUp (last instance)
