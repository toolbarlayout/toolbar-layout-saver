# Toolbar Layout Saver

SketchUp extension to save and restore toolbar layouts.

## Important

**You must quit SketchUp via File > Exit SketchUp for the layout to be saved properly.**
**It must be the last running instance of SketchUp.**

## Installation

1. Copy `toolbarLayoutSaver.rb` into SketchUp's Plugins folder
2. Restart SketchUp

## Usage

### Menu
```
View -> Toolbar Layout
  |- Save Layout
  `- Restore Layout
```

1. Arrange your toolbars as desired
2. Go to **View → Toolbar Layout → Save Layout**
3. To save properly, quit SketchUp via **File → Exit SketchUp** — it must be the **last running instance** of SketchUp
4. Next time you open SketchUp, go to **View → Toolbar Layout → Restore Layout**
